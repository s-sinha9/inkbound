# 🎯 Course Theme: Build a Scalable E-Commerce Backend

## 🧩 Core Project: “InkBound”


### 🏗️ Phase 1 (Monolith)

* Product catalog
* User management
* Order management
* REST APIs
* Database integration

### ⚙️ Phase 2 (Microservices)

* Split into services:

  * User Service
  * Product Service
  * Order Service
  * Payment Service
* Add API Gateway, Service Discovery

### 🚀 Phase 3 (Production-ready)

* Security (JWT/OAuth2)
* Event-driven architecture (Kafka)
* Docker + Kubernetes deployment

---

# ⏱️ Total Duration: ~9–10 Hours

---

# 📚 Course Outline

---

## 🟢 Chapter 1: Spring Boot Fundamentals (1 hr)

**Goal:** Understand basics and run your first app

### Topics:

* What is Spring Boot
* Project setup (Spring Initializr)
* Project structure
* `@SpringBootApplication`
* Embedded server (Tomcat)
* Running your first REST API

### Mini Task:

* Create `/hello` API

---

## 🟢 Chapter 2: Building REST APIs (1 hr)

**Goal:** Create real endpoints

### Topics:

* `@RestController`, `@RequestMapping`
* CRUD APIs
* Request/Response handling
* DTO vs Entity
* Validation (`@Valid`)

### Project Work:

* Build **Product APIs**

  * Create product
  * Get product
  * Update/delete

---

## 🟢 Chapter 3: Database Integration (1 hr)

**Goal:** Persist data

### Topics:

* Spring Data JPA
* Entities & relationships
* Repositories
* Pagination & sorting
* Transactions

### Project Work:

* Connect to DB (MySQL/Postgres)
* Store products & users

---

## 🟢 Chapter 4: Layered Architecture & Best Practices (45 min)

**Goal:** Write clean, scalable code

### Topics:

* Controller → Service → Repository
* Dependency Injection
* Exception handling (`@ControllerAdvice`)
* Logging

### Project Work:

* Refactor into clean architecture

---

## 🟢 Chapter 5: Spring Boot Advanced Features (45 min)

**Goal:** Make app production-ready

### Topics:

* Profiles (`dev`, `prod`)
* Config management
* Actuator
* Caching (Redis intro)

---

## 🟡 Chapter 6: Introduction to Microservices (1 hr)

**Goal:** Break monolith into services

### Topics:

* What is microservices architecture
* Service boundaries
* Inter-service communication

### Project Work:

* Split into:

  * Product Service
  * User Service
  * Order Service

---

## 🟡 Chapter 7: Service Communication (1 hr)

**Goal:** Connect services

### Topics:

* REST communication
* Feign Client
* Error handling between services

### Project Work:

* Order Service calls Product & User Service

---

## 🟡 Chapter 8: Service Discovery & API Gateway (1 hr)

**Goal:** Make system dynamic

### Topics:

* Eureka
* API Gateway (Spring Cloud Gateway)
* Routing

### Project Work:

* Register services
* Route via gateway

---

## 🔴 Chapter 9: Security (JWT + OAuth2) (1 hr)

**Goal:** Secure APIs

### Topics:

* Spring Security
* JWT authentication
* Role-based access
* OAuth2 basics

### Project Work:

* Secure endpoints
* Login API → JWT token

---

## 🔴 Chapter 10: Event-Driven Architecture (45 min)

**Goal:** Decouple services

### Topics:

* Async communication
* Apache Kafka basics
* Producer/Consumer

### Project Work:

* Order placed → event → Payment Service

---

## 🔴 Chapter 11: Dockerization (30 min)

**Goal:** Containerize apps

### Topics:

* Docker basics
* Dockerfile
* Multi-service setup

### Project Work:

* Dockerize all services

---

## 🔴 Chapter 12: Kubernetes Deployment (30 min)

**Goal:** Deploy like production

### Topics:

* Kubernetes basics
* Deployment & Service YAML
* Scaling

### Project Work:

* Deploy microservices cluster

---

## 🔴 Chapter 13: Performance & Scaling (30 min)

**Goal:** Real-world readiness

### Topics:

* Caching strategies
* Load balancing
* Rate limiting
* Observability basics
